Vuetify
Vuetify의 주요 특징은 Vue.js 기반의 Material Design 컴포넌트 프레임워크라는 점입니다. 구글 Material Design 가이드라인을 따르며, 다양한 UI 컴포넌트와 기능을 제공하여 개발 생산성을 높이고, 반응형 웹 애플리케이션을 쉽게 구축할 수 있게 해줍니다
Vuetify 도 2와 3가 있는데 3가 타입 스크립트를 적극 지원한다고 하니 3로 진행.
Vuetify3는 vue cli 와 vite 두개 존재 두개다 preview 이고 디테일한 것들을 이해하기엔 사전지식이 모자라니 vite 선택

vue router
Vue Router는 Vue 애플리케이션의 동적이고 중첩된 라우팅, SPA 구현 지원, URL에 따른 컴포넌트 매핑, 내비게이션 제어 기능, 그리고 스크롤 관리 등을 수행합니다. 
-> 스크롤 관리 기능 관련해서는 https://www.heropy.dev/p/2Hstmu 
vue router history mode 
Vue Router의 history 모드는 URL에서 # 해시 기호를 제거하여 깨끗하고 일반적인 URL을 제공합니다. 

Vuex
Vuex의 특징은 중앙 집중식 저장소로 애플리케이션의 상태를 한 곳에서 관리하며, 예측 가능한 상태 변경과 Vue Devtools와의 통합을 통해 디버깅을 용이하게 합니다. 또한, 단일 상태 트리를 사용하여 전체 상태를 하나의 객체로 관리하고, **동기적인 상태 변경(Mutations)**과 **비동기적 처리를 포함한 로직(Actions)**을 분리하는 등 특징을 가집니다.
